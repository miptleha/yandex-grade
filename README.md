# Обзор Яндекс Грейд

Корпоративное решение для прокачки навыков сотрудников.  
В отличии от [Яндекс Практикума](practicum.yandex.ru) (каждый курс в котором длится от нескольких месяцев до нескольких лет), курсы в [Яндекс Грейде](practicum.yandex.ru/grade) предельно короткие (от нескольких минут до нескольких часов).  
Есть ли что-то полезное в таких коротких курсах? Давайте разбираться.
